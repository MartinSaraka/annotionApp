# AnnotAid

> Simple Design, Advanced AI Precision

An Electron application with React and TypeScript.

## Project Setup

### Install

```bash
yarn
```

### Development

```bash
yarn dev
```

### Build

```bash
# For Windows
yarn build:win

# For MacOS
yarn build:mac-arm
yarn build:mac-intel

# For Linux
yarn build:linux
```
